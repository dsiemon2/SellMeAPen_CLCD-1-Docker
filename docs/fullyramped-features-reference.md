# FullyRamped Feature Reference

Reference document based on [FullyRamped Pricing](https://fullyramped.com/pricing) - a competitor in the AI sales roleplay space.

---

## Feature Breakdown

### 1. Unlimited Custom AI Role-plays (Multi-lingual)

**What it is:** Create unlimited custom sales scenarios where AI plays the customer/prospect role.

**Capabilities:**
- Design custom buyer personas with specific objections, pain points, and behaviors
- Set industry-specific contexts (SaaS, healthcare, finance, etc.)
- Configure difficulty levels and buying signals
- Multi-lingual support allows training in any language (Spanish, French, German, Mandarin, etc.)
- AI adapts responses based on the rep's approach - not scripted

**Use cases:**
- Onboard new sales reps with realistic practice
- Prepare for specific upcoming calls
- Practice handling different buyer types
- Train teams across global offices in their native languages

---

### 2. AI Cold Calls, Warm Calls, Discovery Calls, and More

**What it is:** Pre-built call scenario templates for different stages of the sales process.

**Call Types:**
- **Cold calls**: First outreach, getting past gatekeepers, booking meetings
- **Warm calls**: Following up on inbound leads, demos, or referrals
- **Discovery calls**: Uncovering needs, qualifying prospects, building rapport
- **Demo calls**: Product presentations, handling technical questions
- **Closing calls**: Negotiation, overcoming final objections, asking for the sale
- **Renewal/Upsell calls**: Expanding accounts, preventing churn

**Benefits:**
- Reps practice the specific call type they struggle with
- Managers can assign targeted practice based on performance gaps
- Consistent methodology across the team

---

### 3. AI Coach

**What it is:** Real-time or post-call AI-powered feedback on sales performance.

**Capabilities:**
- Analyzes talk-to-listen ratio
- Identifies missed opportunities (questions not asked, objections not addressed)
- Evaluates tone, pacing, and confidence
- Suggests specific improvements with examples
- Tracks improvement over time
- Provides instant feedback after practice sessions

**Coaching areas:**
- Discovery questioning technique
- Objection handling effectiveness
- Closing behavior and timing
- Value proposition delivery
- Active listening indicators

---

### 4. Team Performance Analytics

**What it is:** Dashboard showing sales team training metrics and performance trends.

**Metrics tracked:**
- Number of practice sessions per rep
- Average scores by call type
- Improvement trends over time
- Skill gap identification
- Completion rates for assigned training
- Leaderboards and rankings

**Management benefits:**
- Identify who needs more coaching
- Track ROI of training investment
- Benchmark team against industry standards
- Spot patterns in common weaknesses
- Data-driven coaching conversations

---

### 5. Unlimited Custom AI Scorecards

**What it is:** Create custom evaluation criteria to score sales conversations.

**Capabilities:**
- Define your own scoring categories and weights
- Align scorecards to your sales methodology (MEDDIC, SPIN, Challenger, etc.)
- Create different scorecards for different call types
- Score on specific behaviors: "Did they ask about budget?", "Did they handle the price objection?"
- Binary (yes/no) or scaled (1-5) scoring options

**Examples:**
- Discovery scorecard: Scored on pain identification, timeline, authority mapping
- Demo scorecard: Scored on personalization, feature-benefit linking, next steps
- Closing scorecard: Scored on urgency creation, objection handling, ask confidence

---

### 6. Real-Call Analysis with AI Scoring

**What it is:** Upload or integrate actual sales calls and have AI score them against your criteria.

**How it works:**
1. Connect to call recording platform (Gong, Chorus, etc.) or upload recordings
2. AI transcribes and analyzes the conversation
3. Scores the call against your custom scorecards
4. Highlights specific moments (good and bad)
5. Generates coaching recommendations

**Benefits:**
- Move beyond practice to real performance measurement
- Consistent, unbiased call scoring at scale
- Managers can review more calls with less time
- Identify coaching moments with timestamps
- Track real-world improvement, not just practice scores

---

### 7. Custom Integrations

**What it is:** Connect the platform to your existing sales tech stack.

**Common integrations:**
- **CRM**: Salesforce, HubSpot, Pipedrive - sync rep data, pull prospect info
- **Call recording**: Gong, Chorus, CallRail - import real calls for analysis
- **Sales engagement**: Salesloft, Outreach - trigger training based on activity
- **LMS**: Existing learning systems - unified training experience
- **HRIS**: Workday, BambooHR - auto-provision users
- **BI tools**: Export data to Tableau, Looker for custom reporting

**Enterprise benefits:**
- Build custom workflows (e.g., auto-assign training when rep loses a deal)
- Single source of truth for sales performance
- API access for custom applications

---

### 8. SAML and SCIM

**What it is:** Enterprise-grade authentication and user provisioning.

**SAML (Security Assertion Markup Language):**
- Single Sign-On (SSO) with your identity provider (Okta, Azure AD, OneLogin)
- Users log in once and access all tools
- Centralized authentication control
- Enforces corporate password policies

**SCIM (System for Cross-domain Identity Management):**
- Automatic user provisioning/de-provisioning
- When someone joins/leaves in your HR system, they're auto-added/removed
- Sync groups and permissions automatically
- Reduces admin overhead

**Security benefits:**
- No separate passwords to manage
- Immediate access revocation when employees leave
- Audit trail of access

---

### 9. Designated CSM (Customer Success Manager)

**What it is:** A dedicated person responsible for your account's success.

**What they do:**
- Regular check-ins (weekly/monthly) to review adoption and results
- Help configure scenarios, scorecards, and integrations
- Provide best practices and benchmarks
- Escalate issues and ensure fast resolution
- Quarterly business reviews with ROI analysis
- Training for new admins and managers

**Value:**
- Single point of contact (not support tickets)
- Proactive optimization suggestions
- Faster time-to-value
- Advocacy within the vendor organization

---

### 10. Advanced Security Configurations

**What it is:** Enterprise security controls beyond standard offerings.

**Capabilities:**
- **Data residency**: Choose where data is stored (US, EU, specific regions)
- **IP allowlisting**: Restrict access to specific network ranges
- **Audit logging**: Detailed logs of all user actions for compliance
- **Data retention policies**: Custom rules for how long data is kept
- **Encryption controls**: Customer-managed encryption keys (BYOK)
- **Role-based access**: Granular permissions (who can see what data)
- **SOC 2 / HIPAA compliance**: Meet regulatory requirements

**Who needs this:**
- Financial services (regulatory requirements)
- Healthcare (HIPAA compliance)
- Large enterprises (internal security policies)
- Government contractors (data handling requirements)

---

### 11. White-Glove Implementation

**What it is:** Hands-on, customized setup and rollout assistance.

**What's included:**
- **Discovery**: Understand your sales process, methodology, and goals
- **Configuration**: Build custom scenarios, scorecards, and integrations
- **Content creation**: Develop initial roleplay library tailored to your products/market
- **Pilot program**: Test with small group, iterate based on feedback
- **Training**: Admin training, manager training, rep onboarding
- **Rollout support**: Communication templates, launch plan, change management
- **Optimization**: Post-launch tuning based on usage data

**Timeline:** Typically 4-8 weeks depending on complexity

**Value:**
- Faster adoption (reps actually use it)
- Higher ROI (configured correctly from the start)
- Reduced internal resource burden
- Best practices built in

### 5. Unlimited Custom AI Scorecards

**Current state:** No scoring system

**Implementation:**
```
Scorecard builder:
- Define categories (e.g., "Discovery", "Objection Handling")
- Define criteria within categories (e.g., "Asked about budget")
- Set weights for each criterion
- Choose scoring type (yes/no or 1-5 scale)

AI scoring:
- After session, AI evaluates transcript against scorecard criteria
- Returns scores with evidence (quotes from transcript)

New routes:
- GET /admin/scorecards - list scorecards
- POST /admin/scorecards - create scorecard
- POST /api/session/:id/score - score a session
```

**Database:**
```prisma
model Scorecard {
  id          String @id @default(cuid())
  name        String
  scenarioType String? // Optional - apply to specific scenarios
  categories  ScorecardCategory[]
}

model ScorecardCategory {
  id          String @id @default(cuid())
  scorecardId String
  name        String
  weight      Float @default(1.0)
  criteria    ScorecardCriterion[]
}

model ScorecardCriterion {
  id          String @id @default(cuid())
  categoryId  String
  name        String
  description String
  scoreType   String // "binary" or "scale"
  weight      Float @default(1.0)
}
```

**Effort:** Medium-High (3 weeks)

### Real-World Scenario Scripts

Detailed scenario examples ready to implement:

#### SaaS Cold Call - SDR Booking a Demo

**Setup:**
- Product: Project management software ($500/mo)
- Buyer persona: VP of Operations at mid-size company
- Goal: Book a 30-minute demo
- Difficulty: Medium

**AI Customer Persona:**
```
You are Sarah Chen, VP of Operations at a 200-person logistics company.
You're busy and skeptical of cold calls. You currently use spreadsheets
and email to manage projects. Pain points: missed deadlines, no visibility
into team workload, too many status meetings.

Objections to use:
- "I'm really busy right now"
- "We've looked at tools before, they're too complicated"
- "Send me an email"
- "How much does it cost?"

You'll book a demo IF the rep: asks good questions about your current
process, identifies a specific pain point, and doesn't push too hard.
```

**Success criteria:** Demo booked with date/time confirmed

- Buyer persona: 35-year-old with young family

- Difficulty: Easy-Medium

You are Mike Thompson, 35, married with 2 kids (ages 3 and 6).
